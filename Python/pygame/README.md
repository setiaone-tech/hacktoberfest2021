# ARCADE RACING made with pygame

## Requirements and Dependencies
Recommendet Python-Version
- `3.7+`

The only external dependency are
- `PyGame`

## Installation
Install the required external dependencies with

```shell
$ pip install pygame
```

## Usage examples
Getting help:
```shell
$ (python3) main.py
```

Controller:
```shell
- KEY UP
- KEY LEFT
- KEY RIGHT
- KEY DOWN
- ENTER
```
